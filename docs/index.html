Slider2.0


easy . fast . light . responsive

So, welcome back to another chapter of easy and elegant sliding. It has been I while since we release the SuperSimpleSlider and there has been a lot going on. We are proud to present to you today the brand new Slider2.0. It is packed with some nice additions and improvements like fade transition, a description feature and the option to put a link on each slide.
BUT don't be afraid, of course we kept the simplicity and all the other conveniences from the so loved SuperSimpleSlider. Besides the changes in the front end a lot changed under the hood. We improved the performance and by directly inserting a custom jQuery Mobile script we saved you a lot of work and unnecessary pageload.

the setup

1

Download the Slider2.0.min.js from the download section an include it right after the jQuery import:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>  
<script src="js/slider2.0.min.js"></script>

2

Next step: Simply build a div with the eae-slider class, to style it via CSS you should set an id too:

<div class="eaeSlider" id="mainSlider">
    <img src="img/sampleImage1.jpg" />
    <img src="img/sampleImage2.jpg" />
    <img src="img/sampleImage3.jpg" />
    <img src="img/sampleImage4.jpg" />
</div>

3

From here you are good to go. But, there's a lot more to setup if you want. The Slider2.0 has a description and a link function. Just add a eae-link="" or eae-desc="" attribute to the <img/> tag:

<img src="img/sampleImage1.jpg" eae-link="http://saveandclose.de" />
<img src="img/sampleImage1.jpg" eae-desc="This is a description!" />

default settings

changeDuration: 500
changes the transition speed, accepts numbers in ms
changeDelay: 0
changes the delay for the auto-slide function, accepts number in ms, 0 is off
changeMode: 'slide'
changes slide mode, acccepts 'slide' or 'fade'
imageCrop: 'cover'
defines the image-crop for the slider, accepts all standard css values for background-size (e.g. 'cover', 'contain')
imagePos: 'center center'
defines the image position, accepts all standard css values for background-position
descColor: '#696969'
defines font color for image description, accepts all standard css color values
descFontSize: '20px'
defines font size for image description, accepts all standard css font-size values
descWidth:'100%'
defines the width of the description bar, accepts all standard css width values, considers descPadding setting, if set to '' it is set to auto
descPadding: 20
defines the padding fotr the desciption, accepts only numeric input
enableCtrl: 'false'
defines wether the controls are visible or not, accepts 'true' or 'false'
ctrlColor: ''
changes the back color of the controls and the description, accepts all standard css color values
ctrlWidth: '50px'
changes the width of the controls, accepts all standard css width values
ctrlHeight: '100%'
changes the height of the controls, accepts all standard css height values
ctrlLeftUrl: ''
defines the path for the background image of the left control (e.g. 'img/icons/arrow_left')
ctrlRightUrl: ''
defines the path for the background image of the right control (e.g. 'img/icons/right')
enableTouch: true
defines wether touch is enabled or not, accepts 'true' or 'false'
bgColor: 'transparent'
defines the slider background color, visible e.g. if imageCrop is set to 'contain', accepts all standard css color values

change settings

First, you can change these default settings globally. To do so you just add a <script></script> tag after the Slider2.0 import and access the eaeGlobalSettings object to perform your changes:

<script>
    eaeGlobalSettings.changeDuration = 400;
    eaeGlobalSettings.changeDelay = 8000;
    eaeGlobalSettings.enableCtrl = 'true';
</script>

IMPORTANT: while changing settings globally keep in mind that you are using JS, so everything, except numeric values, HAS TO BE IN QUOTES!
Second, you can change the settings individually for each slider. In this case you add the eae-settings="" attribute to the eaeSlider div. Seperate multiple statements with a semicolon.

<div class="eaeSlider" eae-settings="enableTouch:false;changeDelay:8000">
    <img src="img/sampleImage1.jpg" />
    <img src="img/sampleImage2.jpg" />
    <img src="img/sampleImage3.jpg" />
</div>

IMPORTANT: while changing settings individually make sure to AVOID USING QUOTES in any way!
